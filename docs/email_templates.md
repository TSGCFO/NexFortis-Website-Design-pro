## Template 5 — QBM Creation Help

**Subject:** Re: Your QuickBooks Order #[ORDER-ID] — Wrong File Type

---

Hi [CUSTOMER NAME],

We received your uploaded file, but it appears to be a **[.QBB / .QBW / other]** file rather than a **.QBM** (Portable Company File), which is what our conversion service requires.

**How to create a .QBM file:**

1. Open your company file in **QuickBooks Enterprise**
2. Go to **File** → **Create Copy**
3. Select **Portable company file (.QBM)** — do NOT select "Backup copy" (.QBB)
4. Click **Next** and choose a save location
5. QuickBooks will create a `.QBM` file (typically 5-30 MB)
6. Upload the `.QBM` file to our portal

**Common mistakes to avoid:**
- Do NOT upload a `.QBB` file (backup) — these are much larger and in a different format
- Do NOT upload a `.QBW` file (working file) — these cannot be transferred safely
- Do NOT upload a `.QBO` file (QuickBooks Online format) — our service is for Desktop Enterprise

**Your order has NOT been refunded** — once you upload the correct `.QBM` file, we will process your conversion immediately.

If you need any help creating the QBM file, please reply to this email and we'll walk you through it.
